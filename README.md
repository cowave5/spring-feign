## http-client

HTTP客户端，参考Netflix Feign实现，使用的Apache Httpclient进行调用

- 依赖

```xml
<dependency>
    <groupId>com.cowave.commons</groupId>
    <artifactId>http-client</artifactId>
    <version>2.7.5</version>
</dependency>
```
